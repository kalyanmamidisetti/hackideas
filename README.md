## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

https://github.com/kalyanmamidisetti/hackideas.git

Installation:

`npm install`

To Run Test Suite:

`npm test`

To Start Server:

`npm start`

To Visit App:

`localhost:3000`
